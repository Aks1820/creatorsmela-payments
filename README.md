# CreatorsMela – Payments & Payouts Dashboard

A responsive Payments & Payouts dashboard built as part of the CreatorsMela Product Challenge. The dashboard provides creators with a clear overview of their earnings, payouts, transactions, and invoices.

## Live Demo

https://agent-6a410e89bea7745bbd92--creatorsmela-payment.netlify.app/

https://github.com/your-username/your-repository

## Features

- Earnings summary
- Earnings overview chart
- Pending payouts
- Completed payouts
- Transaction history
- Payout method overview
- Invoices and statements
- Date and campaign filters
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router DOM
- Recharts

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Design Goals

The dashboard focuses on:

- Clarity
- Transparency
- Clean financial UI
- Responsive layout
- Easy navigation
- Readable information hierarchy

## Notes

- All earnings, payouts, transactions, invoices, and payout methods are mock data created for demonstration purposes.
- No real banking or payment information is collected or stored.
