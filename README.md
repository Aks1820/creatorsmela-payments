# CreatorsMela – Payments & Payouts Screen

A clean, simple payments dashboard for creators to track earnings, payouts, and invoices.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Recharts
- React Router

## Features
- Earnings summary
- Earnings overview chart
- Pending & completed payouts
- Transaction history
- Payout method
- Invoice list
- Date & campaign filters
- Responsive design

## Setup

npm install
npm run dev

## Build

npm run build
